# 🧠 ResumeIQ

### Resume Analysis & Career Intelligence Platform

ResumeIQ is a smart resume analysis tool that helps users evaluate and improve their resumes through ATS scoring, job-role matching, and resume optimization.

---

## 🚀 Features

- 📄 PDF Resume Parsing
- 📊 ATS Score Analysis
- 🎯 Job Role Matching
- 🧠 Skill Detection Engine
- ✨ Resume Optimization
- 🚀 Interactive Gradio Dashboard
