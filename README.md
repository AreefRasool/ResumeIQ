# ResumeIQ 🚀

AI-powered ATS Resume Analyzer built with Google Gemini and Gradio.

## 📌 Features
- Upload PDF resume  
- Extract text automatically using PyMuPDF  
- ATS score generation (/100)  
- Strengths & weaknesses analysis  
- Missing skills detection  
- Improvement suggestions  

## 🧠 Tech Stack
- Python  
- Gradio  
- PyMuPDF (fitz)  
- Google Gemini API  

## ⚙️ How it works
1. Upload your resume (PDF)  
2. AI extracts text  
3. Gemini analyzes ATS compatibility  
4. You get instant detailed report  

## 🚀 Run Locally
pip install -r requirements.txt  
python app.py  

## 🔑 API Key Setup
genai.configure(api_key="YOUR_API_KEY")
